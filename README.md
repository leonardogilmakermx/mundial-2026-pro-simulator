# World Cup 2026 - Statistical Simulator Pro
**Desarrollado por: Ing. G. Leonardo Dominguez Garcia**

Este proyecto es una herramienta de análisis predictivo para la Copa del Mundo 2026, combinando ingeniería de datos y estética deportiva inmersiva.

## 🚀 Características Técnicas
- **Algoritmo ELO Predictivo:** Implementación de la fórmula oficial de la FIFA para el cálculo de probabilidades de victoria entre selecciones.
- **Simulación Montecarlo:** Procesamiento estocástico de 1,000 iteraciones por ejecución para obtener distribuciones de probabilidad robustas.
- **Ajuste de Localía:** Factor de ponderación táctica de +100 puntos ELO para las naciones anfitrionas (México, EE. UU. y Canadá).
- **Interfaz Stadium-Pro:** Diseño UI/UX optimizado para visualización de datos complejos en un entorno temático de estadio.

## 🛠️ Stack Tecnológico
- **HTML5/CSS3:** Estructura y diseño "Stadium Aesthetic".
- **JavaScript (ES6+):** Motor matemático y lógica de simulación.
- **Plotly.js:** Visualización dinámica de histogramas de rendimiento.
- **html2canvas:** Generación de tickets conmemorativos en alta resolución.

## 📊 Objetivo
Proyecto con fines informativos y educativos para demostrar la aplicación de modelos matemáticos en el análisis deportivo (Sports Analytics).
